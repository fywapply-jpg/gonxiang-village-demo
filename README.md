# 供享村社 · 乡村数字经济平台 Demo

> 交互原型 · 产品设计实践 · 2025

一个模拟微信小程序 UI 的 Web 端交互 Demo，用于产品设计作品集展示。

**Live Demo**: https://\<fywapply-jpg\>.github.io/gonxiang-village-demo/

## 功能模块

- **首页** — 村庄 Banner、公告栏（点击切换）、8格功能入口、商品/团购预览
- **供享小店** — 实时搜索商品、点击查看详情
- **惠民团购** — 实时成团进度、参与后计数更新
- **党建助农** — 活动展开/收起、报名交互
- **我的** — 三账户体系（现金/积分/贡献值）、账单明细、利益共享金展示

## 本地运行

```bash
npm install
npm run dev
```

## 技术栈

React 18 · TypeScript · Vite · 纯 CSS-in-JS（无第三方 UI 库）
